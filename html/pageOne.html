1989

Tim Berners-Lee proposed the concept of the World Wide Web at CERN in Switzerland.

1991

Tim Berners-Lee created the first HTML specification, which included 18 elements and the concept of hyperlinks.

1993

Mosaic, the first popular web browser, was released.

1995

HTML 2.0 was released with support for tables and forms.

1997

HTML 3.2 was released with support for frames and style sheets.

1998

The W3C released HTML 4.0, which introduced Cascading Style Sheets (CSS) and JavaScript.

2000

HTML 4.01 was released with minor changes.

2008

HTML 5 was released with support for video, audio, and canvas.

2014

HTML 5.1 was released with minor changes.

2019

HTML 5.2 was released with minor changes.
